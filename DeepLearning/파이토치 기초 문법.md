## 2.1 텐서 다루기
### 텐서 생성 및 변환

텐서는 파이토치의 가장 기본이 되는 데이터 구조

```
import torch
print(torch.tensor([[1,2],[3,4]]))
print(torch.tensor([[1,2], [3,4]], device = "cuda:0"))
print(torch.tensor([[1,2],[3,4]], dtype = torch.float64))
```
